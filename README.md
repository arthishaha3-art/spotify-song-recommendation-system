Spotify Songs Genre Segmentation & Recommendation System 🎵

📌 Project Description

This project uses Machine Learning to analyze Spotify songs and group them into clusters based on audio features such as danceability, energy, loudness, tempo, and more.
It also recommends similar songs based on clustering results.

🎯 Objectives

- Analyze Spotify song audio features
- Perform genre segmentation using K-Means clustering
- Visualize song distributions and correlations
- Build a simple recommendation system

🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

📊 Features

- Dataset analysis and visualization
- Correlation heatmap
- Elbow method to find optimal clusters
- K-Means clustering for genre segmentation
- PCA visualization of clusters
- Song recommendation system

📂 Project Files

- spotify_project.py → Main Python project code
- spotify_songs.csv → Dataset
- spotify_songs_with_clusters.csv → Output dataset with clusters
- Graph images (PNG files) → Data visualization results

▶ How to Run the Project

1. Install required libraries:
   pip install pandas matplotlib seaborn scikit-learn streamlit

2. Run the project using:
   streamlit run spotify_project.py

3. Open the browser link shown in the terminal.

📈 Output

The project generates:

- Genre distribution graph
- Audio feature histograms
- Correlation matrix heatmap
- Elbow method graph
- Cluster visualization using PCA
- Song recommendations
